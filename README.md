## About

Full-stack engineer focused on e-commerce and data engineering. 34 repositories, ~2,400 commits over 14 months — from PHP modules to TypeScript enterprise platforms and AI-powered search.

**OpenCart ecosystem** — search engine (BM25, trigrams, SymSpell, MMR), cross-sell/upsell, Google Merchant Feed, shipping integrations (Nova Poshta, Meest Express), CRM sync (SalesDrive), inventory management, order analytics.

**Data engineering** — retail analytics pipelines (GFK, BigQuery, OLAP/Analysis Services), ETL, PowerPivot, T-SQL.

**Infrastructure** — Proxmox, LXC, Docker, Coolify, self-hosted stack (WireGuard, Grafana, Home Assistant).

Open source contributor — [OpenClaw](https://github.com/openclaw/openclaw) AI agent platform. Based in Ukraine.
